# %格式化

| 占位符 | 替换内容   |
|-----|--------|
| %d  | 整数     |
| %f  | 浮点数    |
| %s  | 字符串    |
| %x  | 十六进制整数 |

格式化整数和浮点数还可以指定是否补0和整数与小数的位数：

```py
print('%2d-%02d' % (3, 1))
print('%.2f' % 3.1415926)
```

# format()字符串格式化

格式化字符串的方法是使用字符串的format()方法，它会用传入的参数依次替换字符串内的占位符{0}、{1}：

```py
>>> 'Hello, {0}, 成绩提升了 {1:.1f}%'.format('小明', 17.125)
'Hello, 小明, 成绩提升了 17.1%'
```
